Projects

01

FaultClaw

Autonomous multi-agent hardware verification pipeline that ingests an RTL module, generates adversarial test cases, and runs them against a golden reference model — producing structured coverage reports and failure diagnoses without human intervention. Built on NVIDIA Nemotron running on Brev cloud compute.

Agent 01

Spec Reader

Parses RTL interface and behavior into a structured internal representation.

Agent 02

Test Generator

Reasons about design assumptions to generate functional, edge-case, and adversarial tests.

Agent 03

Verification Judge

Executes tests, logs failures, computes coverage, and triggers follow-up suites on detected failures.

Breakdown Mode

Full adversarial testing mode that systematically dismantles design assumptions through overflow maximization, boundary stressing, and chained input sequences engineered to expose hidden failure states.
